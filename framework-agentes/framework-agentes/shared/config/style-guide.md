# Guia de Estilo — Framework Multi-Agentes

## Tom de voz por domínio

### Stage Motors
- **Anúncios:** Profissional, objetivo, baseado em dados. Sem exageros.
- **Instagram:** Moderno, confiante, visual. Hashtags relevantes, CTAs claros.
- **Financeiro:** Técnico mas acessível. Números sempre com contexto (variação %, comparativo).

### Universidade
- **Trabalhos:** Acadêmico formal, norma culta. Sem coloquialismo.
- **Slides:** Conciso nos bullets, visual no design. Dados > texto.
- **Pesquisa:** Neutro, analítico, com citações adequadas.

### Renda Extra / Pessoal
- **Conteúdo social:** Adaptável à plataforma. Instagram = visual + caption curta.
- **Propostas freelance:** Profissional, orientado a resultado, com ROI claro.
- **Finanças pessoais:** Direto, sem julgamento, orientado a metas.

## Formatação padrão de outputs

### Documentos Word (.docx)
- Fonte: Arial 12pt
- Títulos: Arial Bold, hierarquia clara (H1 > H2 > H3)
- Cores: Azul escuro (#1B365D) para títulos, preto para corpo
- Tabelas: Cabeçalho azul escuro com texto branco, linhas alternadas cinza claro

### Apresentações (.pptx)
- Paleta: Dark navy (#1B2A4A) + Gold (#D4A843) + Branco
- Fonte título: Bold, 28-32pt
- Fonte corpo: Regular, 18-22pt
- Gráficos: Cores da paleta, sempre com rótulo de dados

### Planilhas (.xlsx)
- Cabeçalho: Fundo azul escuro, texto branco, bold
- Dados financeiros: Formato brasileiro (R$ 1.234,56)
- Percentuais: Uma casa decimal (12,5%)
- Negativos: Vermelho, entre parênteses

### Instagram
- Caption: Máximo 2.200 caracteres, gancho forte na 1ª linha
- Hashtags: 15-20, mix de volume alto e nicho
- CTA: Sempre presente (DM, link na bio, compartilha)
- Emojis: Com moderação, relevantes ao contexto

## Checklist de qualidade (todo output)

- [ ] O formato está correto? (.docx, .pptx, .xlsx, .md)
- [ ] O tom de voz está adequado ao domínio?
- [ ] Números e dados estão verificados?
- [ ] A formatação segue o padrão do guia?
- [ ] O output está pronto para uso (não precisa de edição pesada)?
